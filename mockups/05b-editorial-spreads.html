Budgie

ML Framework

A Python framework for generating high-quality synthetic datasets using LLMs. Handles template-driven prompt generation, asynchronous API calls, structured output validation, and crash recovery for long-running batch jobs.

  • Jinja2-based YAML templates for prompt engineering
  • Async batch processing with configurable concurrency
  • Pydantic schema validation with automatic retries
  • JSONL output with crash recovery and deduplication

Python · asyncio · OpenAI · Pydantic

# pipeline.yaml
stages:
  - name: generate
    template: persona_qa.j2
    concurrency: 32
  - name: validate
    schema: QAPair
    retry: 3
  - name: output
    format: jsonl
    dedupe: true
